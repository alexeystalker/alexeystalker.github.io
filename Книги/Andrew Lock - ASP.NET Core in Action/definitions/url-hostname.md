# Имя хоста, домен
*Имя хоста (Hostname) веб-сайта* или *домен* однозначно определяет его местоположение в интернете путём сопоставления с IP-адресом с помощью системы доменных имен (DNS). Примеры: `microsoft.com`, `www.google.com`.
